# Sanber-QA-45-Cucumber-Auto
UI Automation Test using WDIO Cucumber

How?
```
npm install
npm init wdio
```

in the folder consists of UI automation tests :
- login.feature
- createcategory.feature

How to open & run test : 
```
npm run wdio
```
